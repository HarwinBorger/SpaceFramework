# Width & Height
#### Define the dimentions of your element in space-units to your elements

- [@width](#width)
- [@height](#height)

## @width

```scss
@include width($width);
```

- **$width** value defined in **space-units**

### Example 1
<!-- tabs:start -->

#### ** How to use **
```scss
.example {
	@include with(1);
}
``` 
